Projet Java : jeu du pendu

Création d'un jeu de pendu dans le cadre d'un cours sur les bases de Java.

Il s'agit d'un jeu où un joueur doit deviner le mot qu'un autre joueur a choisi. En début de partie, le mot est représenté
par des tirets correspondant au nombre de lettres à deviner. 
Par exemple, ------- correspond à un mot de 7 lettres
A chaque tour, le joueur qui devine propose une lettre. Si la lettre est dans le mot, l'autre joueur l'inscrit.
Si la lettre proposée n'est pas dans le mot, le joueur qui a choisi le mot rajoute un élément dans un dessin.
Ce dessin représente un petit bonhomme accroché à une potence.

Le jeu s'arrête quand :
- le joueur qui devine a trouvé le mot entier,
- le dessin du pendu est terminé.

Le jeu créé ici ne prend pas en compte le dessin et le joueur peut proposer des lettres à volonté.
